# ILI9488_Basic

[Elecrow CrowPanel ESP Terminal 3.5" RGB Display](https://www.elecrow.com/esp-terminal-with-esp32-3-5-inch-parallel-480x320-tft-capacitive-touch-display-rgb-by-chip-ili9488.html)
* ILI9488 16-bit parallel
* FT6236 Touch
* esp32s3
* Espressif v5.5.2
* Lvgl v9.4

## Function
Starter project which starts WiFi and Lvgl v9.4.
